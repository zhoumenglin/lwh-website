# LWH Website

基于 Astro + Tailwind CSS 构建的网站项目。

## 技术栈

- [Astro](https://astro.build/) - 静态站点生成框架
- [Tailwind CSS](https://tailwindcss.com/) - 原子化 CSS 框架
- [TypeScript](https://www.typescriptlang.org/) - 类型安全

## 快速开始

```bash
# 安装依赖
npm install

# 本地开发
npm run dev

# 构建生产版本
npm run build
```

## 部署

项目通过 Vercel 进行自动化部署。
